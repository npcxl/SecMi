在线工具：控件与接口对应

  • 区块：SM3 功能在 #sec-sm3（标题「SM3 杂凑」）。
  • SM3 摘要：btn-sm3 → gm_sm3_hash；消息 sm3-msg（UTF-8）；输出 sm3-out（64 hex）。
  • HMAC-SM3：btn-sm3-hmac → gm_sm3_hmac；密钥 sm3-hmac-key、消息 sm3-hmac-msg；输出 sm3-hmac-out。

1. cwrap 声明

在 Module 就绪后声明；本页 2 个导出符号。

// 摘要与 HMAC（与页底脚本一致）
var sm3fn = M.cwrap('gm_sm3_hash', null, ['number', 'number', 'number']);
var sm3hmacFn = M.cwrap('gm_sm3_hmac', null, ['number', 'number', 'number', 'number', 'number']);

2. SM3 摘要 gm_sm3_hash

消息为 UTF-8 字节；输出 32 字节摘要，展示为 64 hex。

// 与「计算 SM3」按钮逻辑一致
var msg = new TextEncoder().encode($('sm3-msg').value);
if (msg.length > LARGE) throw new Error('消息过长');
var out = M._malloc(32), inp = ac(msg);
try {
  sm3fn(inp.ptr, inp.len, out);
  var digestHex = u8ToHex(new Uint8Array(M.HEAPU8.buffer, out, 32));
} finally {
  inp.free();
  M._free(out);
}

3. HMAC-SM3 gm_sm3_hmac

密钥与消息均为 UTF-8；输出 32 字节 MAC（64 hex）。

var ku8 = new TextEncoder().encode($('sm3-hmac-key').value);
var mu8 = new TextEncoder().encode($('sm3-hmac-msg').value);
if (ku8.length + mu8.length > LARGE) throw new Error('输入过长');
var out = M._malloc(32), pk = ac(ku8), pm = ac(mu8);
try {
  sm3hmacFn(pk.ptr, pk.len, pm.ptr, pm.len, out);
  var macHex = u8ToHex(new Uint8Array(M.HEAPU8.buffer, out, 32));
} finally {
  pk.free();
  pm.free();
  M._free(out);
}
